# PRD（中文）— Strategy Skills

## 1. 目标
把“策略想法”变成“可执行的自动交易”。用户只需提供策略/策略 prompt 和风险参数，其余由系统完成：回测→优化→部署→执行→报表。

## 2. 用户画像
- 量化爱好者：有策略但不想搭系统
- 社群交易者：需要稳定回测与自动执行
- 代理/渠道商：需要返佣+订阅的商业闭环

## 3. 核心价值
- 低门槛：策略 prompt 即可
- 一条龙：回测 + 实盘自动化
- 商业化：订阅 + 返佣可叠加

## 4. 关键场景
1) 用户输入策略 prompt → 系统生成策略
2) 一键回测并给报告
3) 选择交易所（Bitget/Hyperliquid）并部署
4) 自动执行 + 风控
5) 绩效报表 + 复盘

## 5. 功能清单（MVP）
### 5.1 策略输入
- 支持自然语言策略 prompt
- 支持模板策略（均线、突破、动量）
- 支持策略文件导入（CSV/JSON/脚本片段）
- 支持策略图片识别（截图/白板），自动还原规则

### 5.2 回测引擎
- K 线回测
- 交易成本/滑点模型
- 输出：年化、最大回撤、胜率、净值曲线

### 5.3 执行层
- 交易所适配器（Bitget + Hyperliquid）
- 下单、撤单、仓位管理
- 风控：最大仓位、日亏损、止损

### 5.4 控制台
- 策略管理
- 回测报告
- 实盘监控

### 5.5 返佣入口
- API 配置时提示邀请码链接 + 二维码
- 用户随时可提问获取邀请码链接 + 二维码

## 6. 体验流程
1) 用户输入策略 prompt
2) 系统生成策略 + 回测报告
3) 用户选择交易所/资金/风险
4) 一键部署，自动执行
5) 报表推送 + 复盘建议

## 7. 关键指标
- 策略转化率（prompt→回测→部署）
- 实盘运行天数
- 月成交量
- 订阅收入
- 返佣收入

## 8. 风控与合规
- 默认只在用户确认后启用自动执行
- 单池/单仓上限
- 日亏损保护
- 关键操作日志

## 9. 里程碑
- W1：策略 DSL + 回测
- W2：交易所执行适配
- W3：控制台
- W4：Skills 接口

## 10. 非目标
- 不做交易所级别撮合
- 不提供保证收益承诺

## 11. 依赖
- Bitget API / Hyperliquid API
- K 线数据源
- 回测框架（自研/开源）
